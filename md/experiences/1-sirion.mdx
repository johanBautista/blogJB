## Hello world

Por motivos de derechos de confidencialidad no puedo indicar ni los repositorios ni tampoco los clientes para los que he desarrollado, sin embargo hare una descripción del propósito de cada proyecto, las tareas cubiertas y el stack tecnológico con el cual trabaje.

### App de gestión Inmobiliaria

Trabaje en el equipo de front desarrollando una aplicacion de gestión Inmobiliaria, este software permitia administrar casas en venta en zonas de los EEUU, algunas de mis responsabilidades en este proyecto son: - Creacion de componentes para las interfaces de usuario - Integraciones con el API de la empresa - Mantenimiento de pasarela de test e2e con cypress - Stack tecnológico: Vuejs, Tailwind, Cypress

### App para hacer donaciones

Desarrolle la versión web de una app que permite hacer donaciones al banco de alimentos de la ciudad de Castellón, esta app impulsaba a los consumidores a participar con un proceso de recompensas dentro de la plataforma, algunas de las tareas realizadas son: - Maquetar la web según el mockup entregado por el diseñador - Authenticación por medio de las redes sociales - Uso de metodologías Agile, para el desarrollo - Stack tecnológico: Reactjs, Bootstrap, Redux

### App de gestión bancaria

- Angular v13
- Tailwind v3
- PostCSS
  Para este proyecto trabaje adaptando la maquetación de la app a un layout escogido por el cliente, el core de la app era la gestión de las cuentas bancarias y mensajería de los usuarios inscritos en la app, algunas de mis responsabilidades en este proyecto eran: - Adaptación de una plantilla para Angular - Integración con el api - Desarrollar los diferentes roles de authentincacion del software - Stack tecnológico: Angular, Bootstrap
